# ü§ñ Caramba

## üöÄ Quick Start

### Prerequisites

- Go 1.21 or higher
- Docker and Docker Compose
- API keys for desired providers

### Installation

```bash
go get github.com/theapemachine/caramba
```

### Basic Usage

Start required services:

```bash
docker compose up
```

Run an example:

```bash
go run main.go examples pipeline
```

```go

    ai.WithModel("gpt-4"),
    ai.WithTools([]*provider.Tool{

    os.Getenv("OPENAI_API_KEY"),
```
